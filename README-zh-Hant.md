# Bing-API-NodeJS

[簡體中文](README.md) | [繁體中文](README-zh-Hant.md) | [English](README-en.md)

### 一個基於 Node.js 的自動獲取 Bing 首頁每日更新圖片的 API

支援多種解析度，包括 **1366x768**、**1920x1080 (FHD)** 以及 **3840x2160 (UHD 4K)**，滿足不同場景需求

使用 Node.js 編寫，支援 Vercel 等平台一鍵快速私有化部署
